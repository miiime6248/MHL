[EN] Portable Settings
Configuration files placed here are copied to the application directory(TargetApp) during launch.
Any changes made to these files are retrieved back to the Data folder upon exit.

[KR] 포터블 설정 파일
이곳에 위치한 설정 파일들은 실행 시 애플리케이션 폴더(TargetApp)로 복사됩니다.
해당 파일들의 변경 사항은 종료 시 데이터 폴더로 다시 회수됩니다.
