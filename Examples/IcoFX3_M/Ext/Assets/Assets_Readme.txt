[EN] Extension Assets Plugin
When this plugin is run for the first time, it automatically injects files from the repository into the target application directory(TargetApp).
(Automatic injection is determined by the presence or absence of the Dat folder)

[KR] 확장 리소스 플러그인
이 플러그인은 최초 실행시 저장소의 파일들을 타겟 애플리케이션 폴더(TargetApp)로 자동 주입합니다.
(자동주입은 Dat 폴더의 유무로 판단)
